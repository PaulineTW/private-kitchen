# PRIVATE KITCHEN 

> Private Kitchen is a mobile web application developed in two weeks during the Le Wagon full stack web development bootcamp. 

## Stack
- Back end: Ruby on Rails
- Database: PostgreSQL
- Front end: HTML5, CSS3, JavaScript (ES6), Bootstrap
- Hosting: Heroku
